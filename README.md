Шифр Плейфера для байтов

Программа реализует шифрование и дешифрование с помощью модифицированного алгоритма Плейфера.

Алгоритм работает с блоками по 2 байта и использует матрицу 16x16.


Возможности

- Шифрование текста
- Дешифрование текста
- Шифрование файлов
- Дешифрование файлов
- Работа с русским языком
- Работа с любыми типами файлов
